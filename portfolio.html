BlackJack

This was a final project for one of my classes that I took and decided to add more features to it. This was completely written in Python and uses PyGame to display the BlackJack cards and features. The program uses a terminal to display the rules and asks the user what they would like to do. When they initiate to play the game, PyGame displays the game to the user. There are many more exciting changes coming!
